Bài 1: Nhập môn Excel

  1. Tạo workbook trong Excel

    2. Tạo một workbook:

    • Mở Excel.
    • chọn Blank workbook hoặc nhấn Ctrl+N.

    Nhập dữ liệu:

    • Chọn ô trống, ví dụ A1, rồi gõ text hoặc số.
    • Nhấn Enter hoặc tab để chuyển sang ô tiếp theo.

  2. Lưu workbook vào OneDriver trong Excel

    3. Lưu workbook vào OneDriver:

    • Chịn File -> Save As
    • Nhập tên file và chọn Save

  3. Phân tích định dạng trong Excel

    4. Tự động điền một cột bằng Flash Fill:

    • Trong ô dưới First Name, gõ Molly và nhấn Enter.
    • trong ô tiếp theo, gõ một vài chữ cái trong tên Garret.
    • Khi danh sách các giá trị đề xuất hiện ra, nhấn Return. Chọn Flash Fill Options để có nhiều lựa chọn hơn.

    Tính toán nhanh bằng hàm AutoSum:

    • Chọn ô bên dưới số bạn muốn thêm.
    • Chọn Home -> AutoSum -> The AutoSum.
    • Nhấn Enter.

    Tạo sơ đồ:

    • Chọn dữ liệu muốn hiển thị trong sơ đồ.
    • Chọn nút Quick Analysis ở bên phải phía dưới các ô lựa chọn.
    • Chọn Charts, trỏ chuột qua các lựa chọn có sẵn, rồi tích vào sơ đồ mong muốn.

  4. Hợp tác trong Excel

    5. Chia sẻ workbook với người khác

    • Chọn Share trên ribbon hoặc File -> Share. Lưu ý: nếu chưa lưu tập tin của bạn vào OneDriver, microsoft Excel sẽ nhắc bạn upload file để chia sẻ nó.
    • Chọn người bạn muốn chia sẻ từ menu thả xuống hoặc nhập tên hay địa chỉ email.
    • thêm tin nhắn (tùy chọn) và nhân Send.

  5. Cài đặt Excel trên mobile

    6. Để tải và sử dụng file ở khắp mọi nơi - cơ quan, tại nhà hay trên đường đi, hãy cài đặt Microsoft Excel trên thiết bị di động. Lựa chọn bản cài cho thiết bị của bạn:

    • Microsoft Excel
    • Microsoft Excel cho Android
    • Microsoft Excel cho iOS
